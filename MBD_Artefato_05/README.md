# 🗂️ MBD - Artefato 05

## 📌 Descrição
Este artefato representa a etapa final e consolidada do projeto de banco de dados, reunindo a implementação completa, testes e validações do sistema desenvolvido.

## 🎯 Objetivo
Validar o funcionamento do banco de dados por meio de consultas, testes e possíveis otimizações, garantindo que atenda aos requisitos definidos inicialmente.

## 🛠️ Tecnologias utilizadas
- SQL Server (ou outro SGBD utilizado)
- Linguagem SQL
- Ferramentas de modelagem de dados

## 📊 Conteúdo
- Scripts completos do banco de dados
- Inserção de dados para testes
- Consultas SQL (simples e avançadas)
- Validação das regras de negócio
- Possíveis otimizações

## 🧪 Testes realizados
- Testes de integridade referencial
- Execução de consultas para validação dos dados
- Verificação de consistência das informações

## 🔄 Evolução em relação ao Artefato 04
- Validação completa do banco de dados
- Execução de testes práticos
- Ajustes e melhorias na estrutura e consultas

## ▶️ Como executar
1. Abrir o SGBD (ex: SQL Server)
2. Executar os scripts disponíveis nesta pasta
3. Inserir os dados de teste
4. Rodar as consultas SQL para validação
