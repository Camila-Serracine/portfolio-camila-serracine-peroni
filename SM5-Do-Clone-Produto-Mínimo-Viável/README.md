# 🚀 SM5 - Do Clone ao Produto Mínimo Viável (MVP)

## 📌 Descrição
Este projeto apresenta o desenvolvimento de um produto a partir da análise e replicação de uma solução existente (clone), evoluindo para a criação de um Produto Mínimo Viável (MVP) com funcionalidades essenciais.

## 🎯 Objetivo
Compreender a estrutura de um produto digital, reproduzir suas principais funcionalidades e evoluir a solução para um MVP funcional e simplificado.

- Desenvolvimento de Software
- Análise de Sistemas

## 🧠 Conceitos trabalhados
- Produto Mínimo Viável (MVP)
- Engenharia reversa de produtos
- Priorização de funcionalidades
- Desenvolvimento iterativo

## 📊 Conteúdo
- Análise do produto original (clone)
- Definição das funcionalidades principais
- Desenvolvimento do MVP
- Estrutura do sistema desenvolvido

## 🔄 Etapas do projeto
1. Análise do produto existente
2. Identificação das funcionalidades essenciais
3. Desenvolvimento do clone inicial
4. Refinamento para MVP
5. Testes e ajustes

## 💡 Aprendizados
- Foco no essencial para entrega de valor
- Importância da priorização de funcionalidades
- Visão de produto aplicada ao desenvolvimento
