# 🗂️ MBD - Artefato 04

## 📌 Descrição
Este artefato corresponde à etapa final do desenvolvimento do banco de dados, contemplando a implementação do modelo lógico no SGBD e a criação das estruturas físicas do sistema.

## 🎯 Objetivo
Implementar o banco de dados de forma funcional, garantindo a integridade, consistência e eficiência no armazenamento das informações.

## 🛠️ Tecnologias utilizadas
- SQL Server (ou outro SGBD utilizado)
- Linguagem SQL
- Ferramentas de modelagem (brModelo, draw.io ou similares)

## 📊 Conteúdo
- Criação das tabelas (DDL)
- Definição de chaves primárias e estrangeiras
- Inserção de dados (DML)
- Consultas SQL (SELECT)
- Testes e validações do banco de dados

## 🔄 Evolução em relação ao Artefato 03
- Implementação prática do modelo lógico
- Criação física do banco de dados
- Execução de testes com dados reais ou simulados

## ▶️ Como executar
1. Abrir o SGBD (ex: SQL Server)
2. Executar os scripts SQL presentes nesta pasta
3. Verificar a criação das tabelas e dados inseridos
4. Executar consultas para validação
