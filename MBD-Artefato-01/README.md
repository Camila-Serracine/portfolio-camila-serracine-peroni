# 🗂️ MBD - Artefato 01

## 📌 Descrição
Este artefato faz parte da disciplina de Modelagem de Banco de Dados (MBD) e tem como objetivo apresentar a estrutura inicial do sistema, incluindo a definição de entidades, atributos e possíveis relacionamentos.

## 🎯 Objetivo
Desenvolver a base conceitual do banco de dados, auxiliando na organização das informações e no entendimento das regras de negócio do sistema.

## 🛠️ Ferramentas utilizadas
- Modelagem de Dados
- Diagramas (DER - Diagrama Entidade-Relacionamento)
- Ferramentas de apoio (ex: brModelo, draw.io ou similares)

## 📊 Conteúdo
- Levantamento de requisitos
- Identificação de entidades
- Definição de atributos
- Estrutura inicial do banco de dados

Abra os arquivos disponíveis nesta pasta para visualizar os diagramas e a documentação do projeto.
