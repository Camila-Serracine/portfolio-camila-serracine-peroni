# 📊 MBD - Artefato 02

## 📌 Descrição
Este repositório contém o Artefato 02 da disciplina de Modelagem de Banco de Dados (MBD), com foco na representação e organização estruturada dos dados de um sistema.

---

## 🎯 Objetivo
- Representar a estrutura de dados do sistema  
- Organizar entidades e relacionamentos  
- Facilitar o entendimento do banco de dados  
- Apoiar o desenvolvimento do sistema  

---

## 🧩 Conteúdo
- Modelo Conceitual (DER)  
- Modelo Lógico  
- Relacionamentos entre entidades  
- Regras de negócio  

---

## 🛠️ Tecnologias Utilizadas
- SQL  
- Ferramentas de modelagem de banco de dados (ex: brModelo, MySQL Workbench)
