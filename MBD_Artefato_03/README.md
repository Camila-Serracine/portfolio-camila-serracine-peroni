# 🗂️ MBD - Artefato 03

## 📌 Descrição
Este artefato representa a evolução da modelagem do banco de dados, com foco na transição do modelo conceitual para o modelo lógico, estruturando as tabelas de forma mais próxima da implementação.

## 🎯 Objetivo
Transformar o modelo conceitual em um modelo lógico consistente, preparando o banco de dados para sua criação em um SGBD.

## 🛠️ Ferramentas utilizadas
- Modelagem de Dados
- DER (Diagrama Entidade-Relacionamento)
- SQL (conceitos iniciais)
- Ferramentas como brModelo, draw.io ou similares

## 📊 Conteúdo
- Conversão do modelo conceitual para o modelo lógico
- Definição de tabelas
- Chaves primárias (PK)
- Chaves estrangeiras (FK)
- Normalização dos dados

## 🔄 Evolução em relação ao Artefato 02
- Estruturação das entidades em tabelas
- Implementação de relacionamentos via chaves
- Ajustes para evitar redundância (normalização)
