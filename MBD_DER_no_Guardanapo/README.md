# 📝 MBD - DER no Guardanapo

## 📌 Descrição
Este artefato apresenta a modelagem inicial do banco de dados de forma simplificada, simulando o processo de idealização rápida de um sistema — conhecido como “DER no guardanapo”.

## 🎯 Objetivo
Representar a estrutura básica do banco de dados de maneira intuitiva, facilitando a visualização das entidades e seus relacionamentos antes da modelagem formal.

## 🛠️ Ferramentas utilizadas
- Modelagem conceitual
- Diagramas informais
- Ferramentas de desenho (ou até esboços manuais)

## 📊 Conteúdo
- Identificação inicial das entidades
- Relacionamentos principais
- Estrutura simplificada do sistema
- Representação visual preliminar

## 💡 Importância
Esse tipo de abordagem é comum nas fases iniciais de um projeto, permitindo:
- Rápida validação de ideias
- Comunicação mais simples entre os envolvidos
- Base para evolução da modelagem formal (DER completo)

Abra os arquivos disponíveis nesta pasta para visualizar o esboço da modelagem.
