# 🧪 SM2 - Laboratório de Classificação Visual

## 📌 Descrição
Este projeto foi desenvolvido como uma atividade prática focada na classificação visual de dados, explorando técnicas de análise e interpretação de imagens com apoio de ferramentas de inteligência artificial.

## 🎯 Objetivo
Aplicar conceitos de classificação visual para identificar padrões, categorias ou características em imagens, utilizando abordagens manuais ou assistidas por IA.

## 🛠️ Tecnologias utilizadas
- Inteligência Artificial
- Classificação de Imagens
- Ferramentas de IA generativa ou análise visual

## 🧠 Conceitos trabalhados
- Reconhecimento de padrões
- Classificação supervisionada (quando aplicável)
- Interpretação de dados visuais
- Avaliação de resultados

## 📊 Conteúdo
- Conjunto de imagens analisadas
- Critérios de classificação
- Resultados obtidos
- Comparação entre diferentes abordagens

## 🔍 Metodologia
- Definição das categorias
- Análise das imagens
- Classificação manual ou automatizada
- Validação dos resultados

## 💡 Aprendizados
- Importância da organização dos dados visuais
- Desafios na classificação de imagens
- Uso de IA como apoio na análise

## ▶️ Como visualizar
Abra os arquivos disponíveis nesta pasta para acompanhar o processo e os resultados da atividade.
