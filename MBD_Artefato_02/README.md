# 🗂️ MBD - Artefato 02

## 📌 Descrição
Este artefato dá continuidade ao desenvolvimento do banco de dados, com foco no refinamento da modelagem conceitual e na definição mais detalhada das entidades e seus relacionamentos.

## 🎯 Objetivo
Aprimorar a estrutura do banco de dados a partir dos requisitos levantados, garantindo maior consistência e organização das informações.

## 🛠️ Ferramentas utilizadas
- Modelagem de Dados
- DER (Diagrama Entidade-Relacionamento)
- Ferramentas como brModelo, draw.io ou similares

## 📊 Conteúdo
- Ajustes na modelagem conceitual
- Definição de relacionamentos (1:1, 1:N, N:N)
- Inclusão de novas entidades e atributos
- Regras de negócio aplicadas

## 🔄 Evolução em relação ao Artefato 01
- Refinamento das entidades existentes
- Melhor definição dos relacionamentos
- Correções estruturais no modelo
