# 🤖 SM7 - Engenharia de Software com IA (Bubble.io)

## 📌 Descrição
Este projeto explora o desenvolvimento de software com o uso de inteligência artificial e ferramentas no-code, utilizando a plataforma Bubble.io para criação de aplicações de forma ágil e eficiente.

## 🎯 Objetivo
Demonstrar como a combinação de IA com plataformas no-code pode acelerar o desenvolvimento de sistemas, permitindo a criação de soluções funcionais sem a necessidade de programação tradicional.

## 🛠️ Tecnologias utilizadas
- Bubble.io
- Inteligência Artificial
- Desenvolvimento No-Code

## 🧠 Conceitos trabalhados
- Engenharia de Software com IA
- Desenvolvimento No-Code / Low-Code
- Automação de processos
- Prototipação rápida de aplicações

## 📊 Conteúdo
- Estrutura da aplicação desenvolvida
- Funcionalidades implementadas
- Integração com recursos de IA (quando aplicável)
- Fluxos e lógica do sistema

## 🔄 Metodologia
- Definição do problema
- Planejamento da solução
- Desenvolvimento no Bubble.io
- Testes e ajustes

## 💡 Aprendizados
- Agilidade no desenvolvimento com no-code
- Uso prático de IA em aplicações
- Criação de protótipos funcionais rapidamente

## ▶️ Como acessar
1. Abra o projeto na plataforma Bubble.io (se disponível)
2. Navegue pelas funcionalidades implementadas
3. Teste os fluxos da aplicação
