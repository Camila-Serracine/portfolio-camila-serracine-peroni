# ⚔️ SM3 - Batalha de Modelos: Engenharia de Prompt

## 📌 Descrição
Este projeto apresenta uma análise comparativa entre diferentes modelos de inteligência artificial, com foco na engenharia de prompt e na avaliação da qualidade das respostas geradas.

## 🎯 Objetivo
Investigar como diferentes modelos de IA respondem a um mesmo conjunto de prompts, identificando variações de desempenho, precisão e clareza.

## 🛠️ Tecnologias utilizadas
- Inteligência Artificial
- Engenharia de Prompt
- Modelos de linguagem (LLMs)

## 🧠 Conceitos trabalhados
- Estruturação de prompts
- Comparação entre modelos
- Avaliação de respostas geradas por IA
- Critérios de qualidade (clareza, precisão, relevância)

## 📊 Conteúdo
- Conjunto de prompts utilizados
- Respostas geradas por diferentes modelos
- Análise comparativa
- Pontos fortes e limitações de cada modelo

## ⚖️ Metodologia
- Definição dos prompts
- Execução dos testes em diferentes modelos
- Coleta das respostas
- Análise baseada em critérios definidos

## 🏆 Resultados
- Identificação do modelo com melhor desempenho
- Insights sobre como melhorar prompts
- Diferenças de comportamento entre IAs

## 💡 Aprendizados
- Impacto da engenharia de prompt nos resultados
- Variação de desempenho entre modelos
- Importância de critérios claros de avaliação
