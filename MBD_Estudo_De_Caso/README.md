# 📚 MBD - Estudo de Caso

## 📌 Descrição
Este artefato apresenta o estudo de caso utilizado como base para o desenvolvimento do banco de dados, incluindo o contexto, os requisitos e as regras de negócio do sistema.

## 🎯 Objetivo
Compreender o problema proposto e estruturar as informações necessárias para a modelagem do banco de dados de forma coerente e eficiente.

## 🛠️ Abordagem utilizada
- Análise de requisitos
- Interpretação do cenário proposto
- Identificação de entidades e relacionamentos
- Definição de regras de negócio

## 📊 Conteúdo
- Descrição do cenário
- Levantamento de requisitos
- Regras de negócio
- Identificação dos dados envolvidos
- Base para modelagem (DER)

## 🔗 Relação com os demais artefatos
Este estudo de caso serviu como base para:
- Modelagem conceitual (Artefatos 01 e 02)
- Estrutura lógica (Artefato 03)
- Implementação (Artefatos 04 e 05)
