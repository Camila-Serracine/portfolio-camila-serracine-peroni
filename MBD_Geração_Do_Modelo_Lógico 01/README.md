# 🧩 MBD - Geração do Modelo Lógico 01

## 📌 Descrição
Este artefato apresenta a primeira etapa da geração do modelo lógico do banco de dados, a partir da modelagem conceitual previamente definida.

## 🎯 Objetivo
Converter o modelo conceitual (DER) em uma estrutura lógica, organizando as entidades em tabelas e definindo seus relacionamentos de forma adequada para implementação.

## 🛠️ Ferramentas utilizadas
- Modelagem de Dados
- DER (Diagrama Entidade-Relacionamento)
- Ferramentas como brModelo, draw.io ou similares

## 📊 Conteúdo
- Transformação de entidades em tabelas
- Definição de atributos
- Identificação de chaves primárias (PK)
- Estrutura inicial do modelo lógico

## 🔄 Processo realizado
- Análise do modelo conceitual
- Conversão das entidades
- Organização inicial das tabelas
- Preparação para definição de relacionamentos

## ▶️ Como visualizar
Abra os arquivos disponíveis nesta pasta para visualizar o modelo lógico gerado.
