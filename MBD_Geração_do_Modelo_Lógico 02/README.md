# 🧩 MBD - Geração do Modelo Lógico 02

## 📌 Descrição
Este artefato dá continuidade à construção do modelo lógico do banco de dados, aprofundando a definição das tabelas, relacionamentos e regras de integridade.

## 🎯 Objetivo
Refinar o modelo lógico, garantindo consistência estrutural, integridade dos dados e adequação às regras de negócio definidas no estudo de caso.

## 🛠️ Ferramentas utilizadas
- Modelagem de Dados
- DER (Diagrama Entidade-Relacionamento)
- Ferramentas como brModelo, draw.io ou similares

## 📊 Conteúdo
- Ajustes e refinamentos nas tabelas
- Definição de chaves estrangeiras (FK)
- Implementação de relacionamentos (1:1, 1:N, N:N)
- Aplicação de normalização

## 🔄 Evolução em relação ao Modelo Lógico 01
- Inclusão de relacionamentos completos entre tabelas
- Ajustes para evitar redundâncias
- Melhor organização dos dados
- Adequação às regras de negócio

## ▶️ Como visualizar
Abra os arquivos disponíveis nesta pasta para visualizar o modelo lógico atualizado.
