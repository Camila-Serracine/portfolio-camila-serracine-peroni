# 👥 MBD - Workshop Grupo 08

## 📌 Descrição
Este artefato refere-se ao desenvolvimento de um workshop em grupo, realizado na disciplina de Modelagem de Banco de Dados (MBD), com foco na aplicação prática dos conceitos estudados.

## 🎯 Objetivo
Aplicar, em equipe, os conhecimentos de modelagem de dados para analisar um problema proposto e desenvolver uma solução estruturada utilizando banco de dados.

## 🛠️ Tecnologias e ferramentas
- Modelagem de Dados
- DER (Diagrama Entidade-Relacionamento)
- Ferramentas como brModelo, draw.io ou similares
- SQL (quando aplicável)

## 👥 Trabalho em equipe
Projeto desenvolvido em colaboração com os integrantes do Grupo 08, envolvendo:
- Divisão de tarefas
- Discussão de soluções
- Validação conjunta da modelagem

## 📊 Conteúdo
- Análise do problema proposto
- Definição de entidades e relacionamentos
- Modelagem do banco de dados
- Possível implementação ou simulação

## ▶️ Como visualizar
Abra os arquivos disponíveis nesta pasta para acompanhar o desenvolvimento do workshop.

## 💡 Aprendizados
- Trabalho colaborativo
- Aplicação prática da modelagem de dados
- Resolução de problemas em equipe
